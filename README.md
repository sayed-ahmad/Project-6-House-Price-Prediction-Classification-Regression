# 🏠 House Price Prediction – Classification & Regression Project

## 📌 Overview

This project focuses on building a complete **Machine Learning workflow** to analyze and predict house prices. It is divided into two main phases:

1. **Classification Task:** Predict whether a house is **“Expensive” or “Not Expensive”**
2. **Regression Task:** Predict the **exact house price** in dollars

This allows exploration of both major types of supervised learning: **classification and regression**.

---

## 🎯 Project Objectives

* Build a classification model to categorize houses by price level
* Build a regression model to predict continuous house prices
* Understand how model evaluation differs between classification and regression
* Develop a full end-to-end ML pipeline following industry workflow practices

---

## 🧠 Machine Learning Workflow

### 1. Business & Domain Understanding

* Analyzed key drivers of house prices (location, materials, size, etc.)
* Investigated real estate context and dataset characteristics

### 2. Data Exploration & Cleaning

* Performed exploratory data analysis (EDA)
* Handled missing values and data inconsistencies

### 3. Data Preprocessing

* Imputed missing values
* Encoded categorical variables
* Scaled numerical features
* Feature selection and engineering

### 4. Modeling

* Built classification models (Expensive vs Not Expensive)
* Built regression models (price prediction)
* Compared multiple machine learning algorithms

### 5. Error Analysis

* Evaluated classification performance (accuracy, confusion matrix)
* Evaluated regression performance (MAE, RMSE)
* Analyzed model errors and limitations

### 6. Implementation Strategy

* Designed a reusable ML pipeline
* Structured outputs for reporting and potential deployment
